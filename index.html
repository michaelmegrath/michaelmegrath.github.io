Sudoku

A simple Sudoku app, built in python with pygame. Puzzles that are dynamically generated, and use an unquie algorithm. Includes a tutorial of sudoku strategies, a freeplay mode for creating custom puzzles, and load and save functionallity.

Idyllic

A demonstration of a genetic machine learning algorithm, modelled and inspired by the Youtube channel Code Bullet. Made with C++ using the SDL openGL to render to the screen. On starting the program dots will begin to learn how to navigate to the goal, with each generation improving on the last.
